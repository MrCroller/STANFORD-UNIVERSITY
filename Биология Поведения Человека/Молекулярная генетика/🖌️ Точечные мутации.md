# Text Elements
По 3 буквы (нуклеотидов)
шифрует одну аминокислоту ^TubwLFy2

триплет ^5ubWLx9q

4 возможных буквы ^8DRkaPGp

3 ячейки ^JBtKdrqi

4*4*4=>64 возможных комбинаций ^3przpXCl

Если пара различных триплетов кодирует 1 аминокислоту
они обычно различаются средней из 3 букв ^kPB4kGNI

или ^pQSAL2r7

А вот если мутация с краю,
то могут быть серьезные последствия ^jwazElHc
